# Rotate Slider

Who says sliders always have to move left and right? This one rotates!  
![Rotary slider showing Hitler and senior Nazi commanders like Himmler, Goebbels, etc.](https://github.com/farshadarbab2024/rotate-slider/blob/master/src/assets/images/screenshots/rotate-slider.png)

## Usage

This slider is especially useful for displaying items in landing pages, such as:  
- Team members  
- Customer testimonials or comments  
- Showcasing profiles or portfolios
